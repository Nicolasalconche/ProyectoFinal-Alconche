# PF-Alconche
Proyecto continuado del curso de desarrollo web: Cuenta con una serie de botones modificando el Dom para contar con diferentes secciones a la hora de seleccionar algunos de los mismos, los que cuentan con javascript son: productos y contacto, contacto cuenta con un formulario completamente funcional para contactarse y el de productos utilice como base lo aprendido en el curso con Gonza, me parecio adecuado a lo que yo queria desde un principio que empece desarrollo web, un apartado para mostrar los muebles que tengo para ofrecer, que tambien se puede agregar mas modelos mediante un boton, buscar los que quiero mediante una barra de navegacion y poder ordenarlos alfabeticamante o de mayor o menor precio, los mismos productos se pueden añadir al carrito para comprarlos y tambien sacarlos si es necesario, cuentas con alerts personalizados al agregar un producto nuevo, al agregar al carrito, al finalizar compra, ademas la pagina cuenta con la fecha actualizada y el sitio cuenta con una api de whatsapp para contactarte directamente mediante un boton emergente.
